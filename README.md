✅ Data Cleaning Techniques

✅ Pivot Charts  
✅ Slicers

- 📈 Return Percentage
- 💰 Investment Value
- 🏦 Broker Performance
- 🏭 Sector Analysis
- 📅 Portfolio Trends

✔ Clean raw stock portfolio data

---

# 🧹 Data Cleaning Process

The following cleaning operations were performed:

| ✅ Step | 🛠 Action |
|---|---|
| 1 | Removed blank `Current Price` rows |
| 2 | Filled missing `Sector` values |
| 3 | Removed duplicate records |
| 4 | Converted negative values using `ABS()` |
| 5 | Standardized text formatting |
| 6 | Added calculated columns |
| 7 | Applied conditional formatting |
| 8 | Added data validation |
| 9 | Converted dataset into Excel Table |

---

# 📌 Calculated Columns

| 📊 Column | 📐 Formula |
|---|---|
| Investment Value | `Buy Price × Quantity` |
| Current Value | `Current Price × Quantity` |
| Profit/Loss | `Current Value - Investment Value` |
| Return % | `((Current Price - Buy Price)/Buy Price) × 100` |
