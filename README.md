# 🧹 Data Cleaning Process

Before analysis, several data quality issues were identified and resolved.

### Issues Found

* Missing Current Prices
* Blank Sector Values
* Duplicate Records
* Negative Quantities
* Negative Buy Prices
* Inconsistent Formatting

### Cleaning Techniques Used

| Technique              | Purpose                    |
| ---------------------- | -------------------------- |
| Remove Duplicates      | Eliminate repeated records |
| XLOOKUP                | Fill missing sector values |
| ABS()                  | Convert negative values    |
| TRIM()                 | Remove extra spaces        |
| PROPER()               | Standardize text           |
| Data Validation        | Prevent invalid entries    |
| Conditional Formatting | Highlight anomalies        |

# 📊 Calculated Metrics

The following financial metrics were calculated:

### 💰 Investment Value

```excel
=Buy Price * Quantity
```

### 📈 Current Value

```excel
=Current Price * Quantity
```

### 🟢 Profit / Loss

```excel
=Current Value - Investment Value
```

### 📊 Return Percentage

```excel
=((Current Price-Buy Price)/Buy Price)*100
```
